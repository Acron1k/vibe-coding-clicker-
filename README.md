# Vibecode-Clicker

AI-powered web development clicker game. Click to generate Vibe Codes and become the ultimate AI developer!

![Vibecode Clicker](https://img.shields.io/badge/version-1.0.0-blue)
![React](https://img.shields.io/badge/React-18-61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6)
![Tailwind](https://img.shields.io/badge/Tailwind-3-38BDF8)

## Features

- **Click to Earn**: Generate Vibe Codes (VB) with each click
- **10 AI Tools**: Purchase tools like Claude, GPT, Cursor, and more
- **5 Subscription Tiers**: Upgrade tools from Free to Ultra
- **Upgrades System**: Boost your click power and production
- **Milestones**: Complete achievements for rewards
- **Offline Progress**: Earn while you're away
- **Beautiful UI**: Cyberpunk-inspired neon design

## Tech Stack

- React 18
- TypeScript
- Zustand (State Management)
- Tailwind CSS
- Framer Motion (Animations)
- Vite (Build Tool)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Game Mechanics

### Currencies
- **Vibe Codes (VB)**: Main currency, earned by clicking
- **Dev Points (DP)**: Premium currency for subscriptions
- **Prompt Tokens (PT)**: Super-premium from Pro+ subscriptions

### Tools (3 Tiers)
1. **Basic LLMs**: Claude, ChatGPT, Gemini
2. **IDE Tools**: Cursor, Claude Code, V0, Replit
3. **Premium**: Lovable, Kiro, Anthropic Console

### Subscriptions
- Free → Pro → Pro+ → Max → Ultra
- Higher tiers = more production + special bonuses
